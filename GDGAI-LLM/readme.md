# Build with AI: Tailoring LLMs with RAG and Fine-Tuning

Workshop notebook from the [GDG AI for Science Australia "Build with AI" series](https://gdgaiforscience.github.io/Build-with-AI/session/02-a-finetune.html).

This exercise explored the fundamentals of adapting Large Language Models (LLMs) for domain-specific tasks using the MedQuAD dataset. Topics covered included:

* Dataset preparation and formatting for fine-tuning
* Fine-tuning LLMs using Keras and Hugging Face Transformers
* Model evaluation before and after fine-tuning
* Comparison of Retrieval-Augmented Generation (RAG) and fine-tuning workflows

This repo contains my workshop notebook and serves as a record of concepts and techniques explored during the session.
